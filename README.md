## 서버시작
python3 manage.py runserver 8000
> 자신이 만든 페이지 확인 http://localhost:8000/페이지이름/
>> 태윤이거 더미 페이지 확인 http://localhost:8000/warzone/ <br>
>> 홍일이거 더미 페이지 확인 http://localhost:8000/tft/

## 페이지 만들기
python3 manage.py start app 이름
